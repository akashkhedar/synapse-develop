# Synapse

Synapse is a powerful data annotation platform for machine learning projects. It provides tools for labeling various types of data including text, images, audio, video, and more.

## Features

- **Multi-modal annotation**: Support for text, images, audio, video, and more
- **Customizable labeling interfaces**: Define your own labeling configurations
- **Team collaboration**: Work together with your team on annotation projects
- **ML integration**: Connect with your machine learning models for active learning
- **API access**: Full REST API for programmatic access
- **Self-hosted**: Run on your own infrastructure

## Quick Start

### Installation

```bash
pip install synapse
```

### Running the Server

```bash
synapse
```

Visit `http://localhost:8080` to access the platform.
